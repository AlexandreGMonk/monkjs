🚀 react-native-views

Install with npm

npm install @monkvision/react-native-views @monkvision/corejs @monkvision/react-native --save

Install from yarn

yarn add @monkvision/react-native-views @monkvision/corejs @monkvision/react-native @reduxjs/toolkit react-redux

Note: this module requires react-native-paper for basic use of Monk. For full customization of our components, please refer to @monkvision/react-native

⛓️ Peer dependencies in package.json:

"@monkvision/corejs": "*","@monkvision/react-native": "*",

CameraView#

import { CameraView } from '@monkvision/react-native-views';
