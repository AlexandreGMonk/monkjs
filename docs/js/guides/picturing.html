Taking pictures

The Camera view takes on a more complete state with Sights. We recommend using this <CameraView /> to take pictures of your vehicles.

yarn add @monkvision/react-native-views @monkvision/corejs @monkvision/react-native

Sights Schemes#

A capture scheme is a list of sights [Sight] with a unique string id, a value tetha between 0 and 359 and a string label.

If our AI can work without this metadata, it analyzes much more easily with it. The Camera view then embeds a list of default sights that you can customize in the near future.

  1. { theta: 0, label: "front" }
  2. { theta: 30, label: "front right"}
  3. { theta: 60, label: "front lateral right"}
  4. { theta: 90, label: "middle lateral right"}
  5. { theta: 120, label: "rear lateral right"}
  6. { theta: 150, label: "rear right"}
  7. { theta: 180, label: "rear"}
  8. { theta: 210, label: "rear left"}
  9. { theta: 240, label: "rear lateral left"}
  10. { theta: 270, label: "middle lateral left"}
  11. { theta: 300, label: "front lateral left"}
  12. { theta: 330, label: "front left"}

This scheme will enable a wheel indicator displaying the angle we need to take the picture. Plus an overlay is completing the view showing helping to take position before capture.
